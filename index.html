⚠️ 장애요인 & 해결 방안
🔴 지문 구조 미스매치
영자신문(역피라미드형) vs 수능 영어(논리흐름·빈칸) — 직접 대입 불가.
✅ 해결
AI가 기사 소재로 수능형 지문 재생성. 별도 편집 인력 불필요.
🔴 AI API 운영비용
피드백·음성인식·지문생성 API 비용. 과도 시 가격 경쟁력 훼손 위험.
✅ 해결
배치 처리·1회 생성 후 DB 재활용. 프리미엄 티어만 AI 적용. 추정 월 300~1,500원/인.
🔴 강사·교사 저항
화상영어 강사가 AI를 대체 위협으로 인식, 세션 구조 변경 거부감.
✅ 해결
AI = 사전·사후 자율학습 도구. 강사 영역(라이브)·AI 영역(채점·요약) 명확 분리.
